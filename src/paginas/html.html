Algumas tags principais de html


  1. Header: Não tem nenhuma função vizual, mas ajudar ao motor de busca a saber que todo conteúdo dentro dela é um cabeçalho, um título, um sub-título, uma introdução da página. Tem a facilidade de se fazer estilizações no css usando apenas o nome da tag.

  2. Footer: Também não tem nenhuma função vizual no site, apenas serve para colocar o rodapé do site. Tem a facilidade de se fazer estilizações no css usando apenas o nome da tag.

  3. Nav: Também não tem nenhuma função vizual no site, no entanto, ela serve para dizer aos motores de buscas que seu conteúdo é uma espécie de menu principal do site. Tem a facilidade de se fazer estilizações no css usando apenas o nome da tag.

  4. Section: Serve para separar o site em algumas seções diferentes.

  5. Article: Diz para o motor de busca ou para o scaner que ali tem um conteúdo importante ou principal daquele site.

  6. Aside: É o contrário do article, o conteúdo é interpretado com conteúdo secundário ou menos importante do site.

  7. Time: O navegador entende como um conteúdo relacionado a data, nele pode-se usar o atributo datetime para informar qual data é aquela.

  8. main: É a tag principal de um site, só pode ter uma por site.

  9. address: É usada para se colocar um endereço dentro dela, ou e-mail, ou teefone, qualquer tipo de contato. Normalmente se usa dentro do rodapé (footer)

  10. Figure e figcaption: Usa-se as tags quando se deseja colocar uma foto com uma legenda dentro, a primeira tag usa-se para a foto e a segunda para a legenda.

Todas as tags acima não apresentam nenhuma função vizual no site, elas servem para causar efeitos no buscador ou navegador ou motor de buscas etc. As mesmas são úteis para aplicar formatações no css usando-as sem que seja preciso criar class ou ids para aquele conteúdo.

  9. Validações nativas: São validações feitas nos formulários para se cetificar que o usuário vai responder aos campos de acodo com o pedido pelo mesmo. Existem as validações que podem feitas através do JS, no entanto essa que vamos aprender chama-se nativas por que usam recursos do navegador.

Outras tags html


  1. Tag de vídeo: a tag video permite colocar vídeo no site, deve ser especificado na tag a altura e largura do vídeo, dentro da tag usa-se o atributo source onde se coloca o endereço do vídeo no src e tipo do arquivo do vídeo no type. Para que o vídeo funcione corretamente se usam ainda dois atributos, o autoplay que az o vídeo rodar assim que o usuário entrar na página e o controls para mostrar os controles do vídeo.



  2. Tag de áudio: a tag audio permite colocar audio no site, dentro da tag usa-se o atributo source onde se coloca o endereço do audio no src e tipo do arquivo do audio no type. Para que o audio funcione corretamente se usam ainda alguns atributos, o autoplay que faz o audio rodar assim que o usuário entrar na página, o controls para mostrar os controles do audio, o muted que faz com que o audio esteja mudo dando a opção ao usuário de ativar o som ou não e o preload que faz com a barra de caregamento do audio inicie assim que o usuário entra na página, caso o mesmo tenha pproblemas de internet o áudio já estará carregado.


  3. Tags de citação: As mesmas não tem nenhum efeito visual, mas servem para passar informações aos motores de buscas que ali tem uma tag de citção. As tags são < q>< /q> e < blockquote>< /blockquote>, dentro dessa segunda pode-se usar "cite" para informar um site de onde saiu aquela citação .
