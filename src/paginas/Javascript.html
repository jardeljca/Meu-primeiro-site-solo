Conteúdo aula 5 - Variáveis:

-Para se criar variáveis, usa-se a abreviação "var" antes da mesma. Ex. (var idade = 35), onde a palavra idade é o nome da variável em questão, seguido do sinal de atribuição "=" e por fim do valor da variável.

-Para se usar o nodejs dentro do VSCODE, abre um novo terminal e escreve a palavra node.

-Tipos de dados das variáveis: O javascript não diferencia números inteiro de reais, para ele todos são "number"; string, são cadeias de caracteres, boolean são usados para true or false. Existem muitos outros tipos de dados, mas esses são principais. Usando-se o comando "typeof" o programa retorna informando qual tipo de dado é aquele.

Operadores matemáticos

Operador aritmético Descrição
+ Operador de soma(adição)
- Operador de subtração
* Operador de multiplicação
/ Operador de divisão
** Operador de potência
% Operador de porcentagem, também usado para se obter o resto de uma divisão
Operadores relacionais Descrição
> Operador de maior que
< Operador de menor que
>= Operador de maior ou igual
<= Operador de menor ou igual
== Operador de igualdade
!= Operador de diferente
Operadores de identidade Descrição
=== Operador de identidade ou de igualdade restrita (compara para saber se os valores são de mesmo valor e mesmo tipo)
!== Operador de desigualdade restrita (compara para saber se os valores são de tipo diferente)
Operadores lógicos Descrição
! Operador de negação
&& Operador de Conjunção (e)
|| Operador de disjunção (OU)

Ordem de precedência dos operadores:

  1. () parentes
  2. ** potência
  3. *; /; % Multiplicação, divisão e pocentagem
  4. +/ - Soma e subtração
