Conteúdo aula 5 - Variáveis:

-Para se criar variáveis, usa-se a abreviação "var" antes da mesma. Ex. (var idade = 35), onde a palavra idade é o nome da variável em questão, seguido do sinal de atribuição "=" e por fim do valor da variável.

-Para se usar o nodejs dentro do VSCODE, abre um novo terminal e escreve a palavra node.

-Tipos de dados das variáveis: O javascript não diferencia números inteiro de reais, para ele todos são "number"; string, são cadeias de caracteres, boolean são usados para true or false. Existem muitos outros tipos de dados, mas esses são principais. Usando-se o comando "typeof" o programa retorna informando qual tipo de dado é aquele.
