Anotações sobre o git e github

Alguns comandos do git:

  • git config --global user.name "Nome" - configura o nome de usuário
  • git config --global user.email "email" - configura o email do usuário
  • git config user.name - Mostra o nome do usuário daquele terminal
  • git config user.email - mostra o email do usuário daquele terminal
  • git config --list - Mostra todas as informações do usuário
  • mkdir - Cria uma pasta para o projeto, após digitar mkdir no promp de comando, digita-se o nome do projeto.
