Atributos de estilização css

  1. Características de Bordas:

    1. Quanto ao estilo da borda:
      1. None ou hidden - sem borda
      2. solid - borda sólida

      3. dotted - borda pontilhada

      4. dashed - borda tracejada

      5. double - borda dupla

      6. groove - borda entalhada

      7. ridge - borda em ressalto

      8. inset - borda em baixo relevo

      9. outset - borda em alto relevo

    2. Border radius:
      Arredonda os cantos das bordas de acordo com a porcentagem estabelecida. Podem ser usadas várias unidades de medidas como "px", "%" etc.
      Ex.
      Border radius com 25px

      Border radius com 50px

      Border radius com 50%

  2. Características de cores:

    1. Color: Altera a cor da textos
    2. backgroundcolor: Altera cores de containeres
    3. Degrade: Coloca duas cores ou mais em um container. Se pode ainda adicionar o grau para definir o ângulo de onde as cores vão partir.
      Ex.
      foi aplicado no css o código "background: linear-gradient(140deg, green, yellow, blue, yellow, green);"

    4. Animações: Para fazer a animação, primeiro se cria um pedaço de código no topo do documento como uma espécie de função usando-se o atributo @keyframes seguido do nome da variável e entre chaves os comando da animação que serão execultados, aos comandos que serão executados atribuirá a porcentagens do tempo de animação que será especificado quando for usar no elemento, após chama-se no elemento em que será animado usando o atributo animation-name:variável e se definirá o tempo da animação com o atributo animation:duration:tempo de duração. Tem ainda os atributos animation:delay para esperar um tempo predeterminado para começar e ainda animation-interation-cont: para deerminar a quantidade de vezes que a animação acontecerá. Ex.
      @keyframes exemplo{
      0% {código a ser execultado}
      25% {código a ser execultado}
      50% {código a ser execultado}
      75% {código a ser execultado}
      100% {código a ser execultado}
      }
      .elemento{
      animation-name: exemplo;
      animation-duration: 4s;
      animation-delay: 2s;
      animation-interation-cont: infinite;
      }
    5. Sombras: Para aplicar sombras em um texto ou elementos usa-se o atributo box-shandow: tamanho-lado tamanho-cima-baixo tamanho-sombra cor, para elementos ou text-shandow para textos. Se usar os tamanhos com valores negativos a sombra é projetada para cima e esquerda. Ex.

      SOMBRA

      SOMBRA

    6. Transição: As transições permitem que determinada característica css mude com a execução de algum evento. Ex. quando passo o mouse por cima daquele elemento, ele muda de cor ou tamanho ou outra característica etc.
